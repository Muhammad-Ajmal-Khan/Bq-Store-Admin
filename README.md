# BQ Store - Admin Panel

A fully functional e-commerce admin dashboard built with **Vanilla JavaScript**, **Tailwind CSS**, and **Firebase**.

https://muhammad-ajmal-khan.github.io/Bq-Store-Admin/index.html

## ✨ Features
- **Firebase Authentication:** Secure login for admin access.
- **Product Management:** Add and delete products in real-time.
- **Category Management:** Manage product categories seamlessly.
- **Cloud Firestore:** Real-time data synchronization.
- **Responsive Design:** Built with Tailwind CSS for a professional look.

## 🛠️ Tech Stack
- HTML5 & Tailwind CSS
- Vanilla JavaScript (ES6+)
- Firebase (Authentication & Firestore)
